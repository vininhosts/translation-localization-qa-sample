# English to Brazilian Portuguese Localization QA Sample

Self-initiated portfolio sample for Upwork and freelance translation profiles.

This project demonstrates:

- English to Brazilian Portuguese translation
- software/app localization style
- glossary and terminology consistency
- translation QA notes
- cultural and tone adaptation
- final clean copy ready for product use

The sample is fictional and created only as a portfolio demonstration. It is not client work.

## Files

- `source/source_en.md`: original English product copy
- `deliverables/translation_pt_br.md`: final Brazilian Portuguese translation
- `deliverables/glossary.csv`: terminology choices
- `deliverables/qa_notes.md`: QA notes and translation decisions
- `deliverables/case_study.html`: polished uploadable case study
